# global_housing

Project 2: Macroeconomics & Pricing (Scikit-Learn)
Target: Banks, Real Estate Funds, Finance

The problem I solve: I replace gut-feeling pricing strategies with hard, data-driven mathematical proofs.

Technical implementation: This machine learning pipeline correlates global market data (e.g., mortgage rates, inflation) with asset prices, utilizing a Random Forest Regressor to calculate the exact percentage weight of each economic driver.

Your ROI: Management receives a validated risk and pricing model to forecast market fluctuations and mathematically secure investment or lending decisions.
